Mafia Codes
We will be using google maps to create a point of interest by using long press on map, this will create a marker with the required radius for the geofence. Then we will be using the latest and best #GeofencingClient to create a geofence. The geofence events will be delivered to our own #BroadcastReceiver where we will handle the triggered #geofence. The geofence events will be delivered by #notifications (using notification channels)
